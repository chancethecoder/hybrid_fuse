# hybrid fuse

This is just practice.  

Mount root directories to one mount point.  

## Install

```bash
git clone https://github.com/chancethecoder/hybrid_fuse
cd hybrid_fuse
make
```

## Usage

### mount

```bash
./hb_fuse [mount-point] [rootDir] [rootDir]
```

### unmount

```bash
sudo unmount [mount-point]
```
